# RAMO

Современное кафе в центре Воронежа. MVP включает сайт с QR-меню, админ-панель и мобильное приложение для гостей.

## Репозиторий

- `site/` — сайт и админ-панель на Next.js 16 + React 19 + TypeScript.
- `mobile/` — мобильное приложение для гостей на React Native + Expo.

## Возможности

### Сайт (`site/`)
- QR-меню для заказа за столом (`/menu?tableId=N`)
- Админ-панель: меню, столы, бронирования, заказы, настройки
- Синхронизация меню и стоп-листа с iiko
- Автоматическая синхронизация с iiko каждую ночь в 03:00
- Уведомления о заказах в Telegram
- Карта зала с занятостью столов
- Программа лояльности: начисление баллов, оплата баллами, личный кабинет

### Мобильное приложение (`mobile/`)
- Просмотр меню по категориям
- Корзина и оформление заказа со стола
- Сканер QR-кодов на столах
- Карта зала и бронирование столов
- Личный кабинет с баллами лояльности

## Быстрый старт

### Backend (сайт)

```bash
cd site
npm install
npx prisma generate
npx prisma db push --accept-data-loss
npm run seed
npm run build
npm run start
```

- Сайт: http://localhost:3000
- Админка: http://localhost:3000/admin
- Логин: `admin`
- Пароль: `admin123`

### Мобильное приложение

```bash
cd mobile
npm install
npx expo start
```

Для запуска на реальном устройстве создайте файл `.env` с LAN IP компьютера:

```bash
EXPO_PUBLIC_API_URL=http://192.168.1.100:3000
```

## E2E-тесты

```bash
cd site
npx playwright test
```

## Примечания по iiko

- Для синхронизации с iiko нужен API Login (и опционально Organization ID).
- iiko не предоставляет webhook/push на изменение меню, поэтому используется polling.
- Меню синхронизируется автоматически раз в сутки, стоп-лист — в рамках этой же синхронизации.

## Безопасность

Перед продакшеном обязательно:
- Смените `JWT_SECRET`
- Смените пароль администратора
- Не коммитьте `.env` и `dev.db`
